设计

我们热爱设计，所以我们注重设计。我们在 Piedu 中采用大胆、鲜明的色彩辅以 Material Design 规范中的层次感，在 UnitedHubs 官方网站的设计上采用成熟稳重的经典蓝，在 inQR App 设计中则灵活运用类 Material Design 元素打造与系统视觉相近却又不落俗套的绝佳体验。

我们同样热爱平面设计。在宁波鄞州高级中学 2017 届 9 班的班级形象设计中，我们独树一帜地采用了颜色分割的方法，充满了青春活力又不失格调，衍生设计的班服成为鄞高独特的风景线。

我们为不同的项目大胆尝试不同的设计方案，也将能为您打造不一样的形象。

开发

绿叶清泉自诞生以来就未曾停止过对新技术的执着追求。为了更好的产品，我们紧随 IT 业界潮流，大胆创新，整合各种新技术，为项目注入澎湃动力。

我们的自有项目中， Piedu 采用 Meteor 全栈开发框架开发，是国内少有的 Meteor 生产级应用。 Klass 则在 Meteor 架构上寻找突破，增强了它的渲染性能， SEO 取得成绩。除此之外，我们使用 ES6 ，发扬 HTML5/CSS3 标准，成为最早一批将其用于生产环境的开发者。

同时，我们大力支持开源项目的发展与繁荣。团队创始人之一的吕世博为 Node.js 中文本地化组成员，同时也是 Meteor 中文社区的一员。

交付

没人想要一个还需要你进一步加工，或者加工过度的交付件。我们的 Web 项目都使用 Git Repo + Dockerfile 的模式交付，平面设计类项目都采用 PSD/AI 的方式交付，也可以提供各种格式的导出档，前端设计类都以 zip with index.html 的方式交付，轻松畅快。

进一步地，我们还提供一步到位的 Web 项目直接交付功能。我们可以将应用直接托管至 DaoCloud 云平台，一切你需要做的只是为计算资源持续付费。
